In-class Lab 10.1

  1. Exercise 1

    1. If using your Linux VM, create directory lab10 under your /var/www/labs directory. Otherwise create directory lab10 under the document root of whatever web server you are using. If using XAMPP on your external USB drive it will probably be either: E:\xampp\htdocs or: F:\xampp\htdocs

    2. In jEdit or another text editor, create a PHP script file named lab10x1e1.php that uses the preg_match() function to find patterns in a string.

    3. Declare a string variable and assign it the value:

      "The dog's not the turkey, the turkey's the turkey, you turkey!"

    4. Use preg_match() to search your string for the word "turkey".

    5. If the word is found print an appropriate message, such as: "I found the turkey", otherwise print out a message such as: "I did not find the turkey".

    6. Also, print out the contents of the optional third argument to preg_match(), an array of matched paterns, using the built-in function print_r.

  2. Exercise 2

    1. Create a PHP script named lab10x1e2.php and save it into your lab10 directory.

    2. Again, declare a string variable and assign it the value:

      "The dog's not the turkey, the turkey's the turkey, you turkey!"

    3. This time use preg_match_all() to search your string for the word "turkey".

    4. If the word is found print an appropriate message, such as: "I found the turkey", otherwise print out a message such as: "I did not find the turkey".

    5. Also, print out the contents of the optional third argument to preg_match_all(), an array of matched paterns, using the built-in function print_r.

    6. Test your code and show it to your instructor.
