In-class Lab 5

  1. Exercise 1 - This Old Man script using basic arrays

    1. Create a script named lab5ex1.php and save it into your labs directory.

    2. Add 2 numerically indexed PHP array variables to hold the values that change in the verses of the "This Old Man" song:

      • First array values are: "on my thumb", "on my shoe", "on my knee", "on a door"
      • Second array values are: "one", "two", "three", "four"

    3. Use a for loop and an echo statement to print the song. Your output should look like this:

      This old man, He played one
      He played knick-knack on my thumb
      ...with a knick, knack, paddy-whack
      give a dog a bone
      This old man came rolling home

      This old man, He played two
      He played knick-knack on my shoe
      ...with a knick, knack, paddy-whack
      give a dog a bone
      This old man came rolling home

      This old man, He played three
      He played knick-knack on my knee
      ...with a knick, knack, paddy-whack
      give a dog a bone
      This old man came rolling home

      This old man, He played four
      He played knick-knack on a door
      ...with a knick, knack, paddy-whack
      give a dog a bone
      This old man came rolling home

  2. Exercise 2 - This Old Man script using associative arrays

    1. Create a script named lab5ex2.php and save it into your labs directory.

    2. This time use a single "associative" PHP array for which you specify both keys and values:

      The array contains: "one" => "on my thumb", "two" => "on my shoe",
                                         "three" => "on my knee", "four" => "on a door"

    3. Use a foreach loop and an echo statement to print the song. Your output should look like the Example 1 output.
