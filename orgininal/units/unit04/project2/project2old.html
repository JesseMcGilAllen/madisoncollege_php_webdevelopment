1. Concepts

This project is designed to enhance a simple PHP web application to use arrays and a file. You will learn these concepts:

  • Adding Functionality to a PHP script
  • Creating a PHP script
  • List Data
  • Array Variables
  • Writing data to a text file.
  • Reading data from a text file.

This project will introduce the use of array variables and files with the PHP programming language.

4. Exercises

Create a directory named "project2" under your web server's document root directory on your hard drive. Place all HTML files and PHP scripts for this project in this new directory.

Note: If you are working on an MATC lab computer, always save a copy of this directory and all the files in it to either the class server, a USB thumb drive, a floppy disk or similar media, at the end of each work session. Otherwise your work will be lost!

Exercise 1

  • Download these two files containing an HTML form (p2form.html) and a PHP script for processing the form (p2process.php) into your project2 directory. Be sure to give the files the original names, since the HTML form file invokes the PHP file by name.

  • Modify the PHP script named p2process.php to use an array for product price information and to save order information to a file. Modify the data collected by the form in p2form.html to include customer name and address fields:

    • Create form fields for the customer name and address (just one address field for now) and local variables in the PHP script to store the data from the form's new fields.
    • Modify the PHP script to use an array variable to store product names and prices. Use code similar to that used on page 83 of the text.
    • Display the customer name and address along with other order information already being displayed by the PHP script.
    • If the total number of items ordered is greater than zero, write the customer order to a file named "orders.txt" in your project2 directory.
    • Write the data to the file in tab-delimited format, similar to the way data is written to a file by the Bob's Auto Parts application in the text.
    • Use appropriate file lock and unlock functions to prevent multiple users of your script from writing to the orders file at the same time.

Exercise 2

  • Write a PHP script named p2view.php that reads the orders.txt file in your project2 directory and displays the on a web page:

    • Open orders.txt for reading and display your own appropriate message if the file open statement fails (suppress the default PHP error message).
    • Display the company name, "RPCV of Wisconsin", and the text "Pending Orders" both as a title and as headers in the body of the web page.
    • Lock the file with a shared, read lock before attempting to read data from it.
    • Display each order on a separate line.
    • Do not forget to unlock the file when done reading it, and close the file explicitly.
