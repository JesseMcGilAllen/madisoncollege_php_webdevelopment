  1. Exercise 1

    1. Open a text editor and enter the following in a new file.

      <html>
       <head>
        <title>Personalized Hello World</title>
       </head>
       <body>
        <?php
          if(!empty($_POST['name'])) {
            echo "Greetings, {$_POST['name']}, and welcome.";
          }
        ?>
        <form action="<?php $PHP_SELF; ?>" method="post">
        Enter your name: <input type="text" name="name" />
        <input type="submit" />
       </body>
      </html>

    2. Save the file as lab1x1.php into your document root directory..

    3. Open a browser and enter the URL:

      http://localhost/lab1x1.php

    4. Show the contents of the file and the screen output to your instructor.
