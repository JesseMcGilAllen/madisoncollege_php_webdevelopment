In-class Lab 1.2

  1. Exercise 1 - Log in to MySQL and view some data

    1. If using Linux or Apple OSX, open a terminal window. On Windows use the Putty terminal emulator instead.

    2. Log in to the class MySQL server using ssh (Secure Socket Layer Shell) by issuing a command similar to the one below. Use the user name given to you by your instructor rather than the one used here.

      ssh djefferson@itins3.matcmadison.edu

    3. Enter your password when prompted. Your instructor will provide you with a password.

    4. If using Putty, you will be presented with a dialog box in which you must enter the name or IP address of the server (itins3.matcmadison.edu in this case). Then click Open and you will be prompted for your username and password.

    5. Use a command line similar to the following to log in to MySQL. Use the user name given to you by your instructor rather than the one used here.

      mysql -u djefferson -p

    6. Again you will be prompted for a password. Use the password provided by your instructor.

    7. Enter the following series of commands in MySQL, and answer the questions below:

      USE winestore;
      SHOW TABLES;
      SELECT * FROM region;

    8. How many tables are there in the winestore database? ______

    9. What is the name of region number 7? ______________________


  2. Exercise 2 - Create a table in your own database schema

    1. Change to your personal database (called a schema in some DBMSs) by entering a command similar to the following. Your instructor will tell you the database name to use:

      use djefferson;

    2. Create a table named test using the following statement.

      CREATE TABLE test (
        pk integer NOT NULL,
        val varchar(20) );

    3. Insert a row into the table test by issuing this statement.

      INSERT INTO test (pk, val)
        VALUES (1, 'one');

    4. Insert two more rows into the table using the same syntax as above. Use any values you wish.

    5. Run the statement below and show your instructor the results.

      SELECT * FROM test;

    6. Enter the MySQL command:

      exit;

      to quit MySQL and the Linux command:

      exit

      (no semi-colon) to sign off from the class server.
