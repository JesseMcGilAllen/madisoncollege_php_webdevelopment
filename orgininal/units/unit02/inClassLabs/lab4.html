In-class Lab 4

  1. Exercise 1

    1. Download the following HTML form file:

      • myform.html

      • Save the file as myform.html in the labs directory under your document root directory.

    2. Open myform.html in a text editor and inspect it to determine what fields it contains.

    3. Open myform.html in a browser to see how it works.

  2. Exercise 2

    1. Create a script named myscript.php and save it into your labs directory.

    2. Add PHP variables to hold the values of the fields from the HTML form.

    3. PHP uses the $_POST[ ] array to hold form field values after the form is posted.

    4. For example, if there is a form field called "name", your variable assignment statement will look like:

      $name = $_POST['name'];

    5. Add PHP code blocks embedded in HTML to print the descriptions and values of all fields entered in the myform.html HTML form.

    6. If a form field is empty, print a "no value submitted" message instead of the field value.

    7. Test your code and show it to your instructor.
