In-class Lab 9.1

  1. Exercise 1

    1. If using your Linux VM, create directory lab09 under your /var/www/labs directory, if it does not exist. Otherwise create directory lab09 under the document root of whatever web server you are using. If using XAMPP on your external USB drive it will probably be: E:\xampp\htdocs or: F:\xampp\htdocs

    2. In jEdit or another text editor, create a PHP script file named lab9x1script.php and save it into your lab09 directory.

    3. Use the file_exists() function to test for the existence of a file. If the file does not exist, display a message containing the file name, similar to the openFile.php example in class, and exit your script.

      Note: The directory you read the file from and the file itself must be readable by user "nobody", "apache", "www-data" or whatever user the web server on your computer runs as.

    4. Add PHP code to open the file for reading with the fopen() function.

    5. Add code to read the file one line at a time with the fgets() function until end-of-file is detected by the feof() function.

    6. Then close the file with the fclose() function.

    7. Test your code by first using the name of a file that does not exist. Then switch to the filename of a file you have created (in a text editor) with 3-4 lines of text in it.

    8. Show your instructor that both versions of your code work.
