Project 5 - Arrays

  1. Exercise 1 - Creating Arrays

    1. Create a basic, numerically indexed, array named $items by using the "[ ]" operator. The values of the array elements should be: "milk", "bread", "juice", "hamburger", "sausage"

    2. Use the print_r() function to display the contents of your $items array.

      Note: Use "<pre>" tags so that print_r() displays each element of the array on its own line.

    3. Now create a basic array named $groceries by using the array() function. It should have the same keys and values as array $items.

    4. Use the print_r() function to display the contents of your $groceries array.

  2. Exercise 2 - Associative Arrays

    1. Create an array variable named $properties using the array() function with these key/value pairs: "Color"/"Red", "Style"/"Coupe", "Engine"/"2 liter V6", "Seats"/"Cloth with Leather", "Wheels"/"Aluminum Alloy".

    2. Use the var_dump() function to display the contents of your $properties array. You will want to use "<pre>" tags to enclose the output of var_dump() .

    3. Use the array_keys() function to create an array named $names from the keys of the $properties array. Then use var_dump() to display the contents of your $names array.

    4. Use the array_values() function to create an array named $values from the values of the $properties array. Then use var_dump() to display the contents of your $values array.
